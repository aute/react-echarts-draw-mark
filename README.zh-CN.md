## :rocket: 使用者指南

通过npm下载安装代码

```bash
$ npm install --save react-echarts-draw-mark
```


如果你是webpack等环境

```js
import reactEchartsDrawMark from 'react-echarts-draw-mark';
```

## :kissing_heart: 贡献者指南

首次运行需要先安装依赖

```bash
$ npm install
```

一键打包生成生产代码

```bash
$ npm run build
```

运行单元测试:

```bash
$ npm test
```

修改 package.json 中的版本号，修改 README.md 中的版本号，修改 CHANGELOG.md，然后发布新版

```bash
$ npm run release
```

将新版本发布到npm

```bash
$ npm publish
```
